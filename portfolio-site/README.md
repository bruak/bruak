# Burak Soykan Portfolio Site

Static portfolio website for an R&D Software Developer profile focused on industrial IoT, real-time systems, telemetry pipelines, microservices and production-ready delivery.

## Structure

```text
portfolio-site/
├── index.html
├── styles.css
├── script.js
└── README.md
```

## Local Preview

Open `index.html` directly in a browser, or serve the folder with a simple static server:

```bash
cd portfolio-site
python -m http.server 8080
```

Then open:

```text
http://127.0.0.1:8080
```

## Deployment

This folder is prepared as a static site and can be deployed through GitHub Pages, Netlify, Vercel static output, cPanel `public_html`, Nginx or any basic static hosting provider.

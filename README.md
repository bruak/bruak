## System Design Mindset

I focus on building systems that are:

- **Observable** — logs, metrics and dashboards at every layer
- **Deployable** — Dockerized, reproducible, environment-agnostic
- **Maintainable** — modular structure, clear service boundaries
- **Real-time capable** — WebSocket and MQTT where latency matters
- **Service-oriented** — decoupled components with defined interfaces
- **Production-aware** — designed for reliability, not just functionality

---

## Problems I Like Solving

- How to collect reliable data from heterogeneous machines
- How to normalize machine states into readable production insights
- How to reduce latency in live monitoring dashboards
- How to deploy services reliably on both Linux and Windows environments
- How to turn raw telemetry into operational decisions
- How to bridge legacy industrial protocols with modern web infrastructure
